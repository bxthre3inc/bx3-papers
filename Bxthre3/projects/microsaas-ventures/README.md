## Product Deep Dives

### PaintPro
White-label mobile app for painting contractors:
- Camera + AR tape measure overlay on room photos
- Automatic square footage calculation
- AI estimate builder → sends quote via SMS/email
- $29/mo white-label / $99/mo pro
- **Marketing → Lead qualification → Estimate follow-up → Review collection**
- [→ AGENTS.md](./PRODUCTS/paintpro/AGENTS.md)

### EscapeSlv
At-home team escape room experience:
- Host downloads app, builds cardboard+duct-tape headset
- Each player uses 1 phone (cast to shared screen or mirror)
- 6 themed escape rooms, 60-90 min sessions
- Party booking, digital hints, scoreboard
- **Marketing → Party lead → Booking confirmation → Review + referral**
- [→ AGENTS.md](./PRODUCTS/escapeslv/AGENTS.md)

### TradeDesk
Agentic back office OS for trades (plumbing, electrical, HVAC, locksmith, roofing):
- AI handles: call answering, job scheduling, invoice follow-up, parts procurement
- Customer portal for job tracking
- **Marketing → Contractor acquisition → Setup assist → Ongoing CS**
- [→ AGENTS.md](./PRODUCTS/tradedesk/AGENTS.md)

### Special Delivery
Anything delivery — food, goods, errands — not just food:
- Onboard both restaurants AND general retail stores (grocery, pharmacy, hardware, flowers, etc.)
- Driver fleet for same-day SLV delivery
- Subscription model + per-delivery fees
- **Marketing → Driver/Store recruitment → Order dispatch → CS → Reviews**
- [→ AGENTS.md](./PRODUCTS/special-delivery/AGENTS.md)

### SLV Ride
Uber-style ride-hailing for San Luis Valley:
- Route-capable drivers (rural, longer distances, scheduled rides)
- SafeRide option (verification layers)
- **Marketing → Driver acquisition → Rider campaigns → CS**
- [→ AGENTS.md](./PRODUCTS/slv-ride/AGENTS.md)

### SLV Now
Local news, events, business directory for San Luis Valley:
- AI-curated from: local gov meetings, school events, church bulletins, community boards
- Business directory → paid featured listings
- Event ticketing (small venues, classes, community events)
- **Marketing → Content aggregation → Directory monetization → Event promotion**
- [→ AGENTS.md](./PRODUCTS/slv-now/AGENTS.md)

### AgentDeploy
**Distinct from AgentOS:** AgentDeploy = single-agent-per-role simplicity. AgentOS = coordinated multi-agent mesh.

One agent. One role. Deployed in minutes:
1. Choose role (Sales / CS / Marketing / HR)
2. Connect tools (Gmail, Calendar, CRM, Stripe, Notion, etc.)
3. Upload knowledge base (docs, paste, URLs)
4. Name agent, set tone, define guardrails
5. Run 5 test scenarios
6. Agent goes live
7. Monitor via dashboard

- $49-$199/mo tiered pricing, 14-day free trial
- **Marketing → Trial signup → Onboarding concierge → Go-live → CS**
- [→ AGENTS.md](./PRODUCTS/agentdeploy/AGENTS.md)

## Automated Operations Blueprint

### Department Structure (per product)
Each product has 3 automated departments:
- **Marketing** — Demand gen, content, ads, social, SEO, email campaigns
- **Sales** — Lead qualification, outreach, onboarding, pipeline management
- **Customer Success** — Support tickets, refunds, reviews, retention

### brodiblanco's Role (Human-in-the-Loop)
| Responsibility | What brodiblanco Does | What Agents Do |
|---|---|---|
| Strategy | High-view product direction | Execute |
| Budget | Approve ad spend | Allocate per agent |
| Brand/Voice | Define tone/personality | Apply |
| Pricing | Final pricing decisions | Track/monitor |
| Partnerships | Key partner introductions | Nurture |
| Escalations | P0/P1 decisions | Route up |
| Content approval | Final copy sign-off | Draft |

### Agent Hierarchy
```
growth-command (Department Head)
├── [product]-growth (Marketing Lead)
├── [product]-promo (Campaign Agent)
├── [product]-sales (Sales Lead)
├── [product]-onboarding (Onboarding)
├── [product]-support (CS Lead)
└── [product]-review (Review/Rentention)
```

### Shared Resources Across All Products
- **Gmail** — Unified outreach, templated sequences
- **Airtable** — Lead pipeline, customer records, order tracking
- **Google Calendar** — Campaign scheduling, meeting booking
- **Stripe** — Billing, promo codes, refunds
- **Zo Space** — Customer dashboards per product
