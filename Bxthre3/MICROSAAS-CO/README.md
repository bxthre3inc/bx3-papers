## Agentic Stack Architecture

### Agent Launcher ←→ Agentic Relationship

```
┌─────────────────────────────────────────┐
│           Agentic (full mesh)            │
│  Multiple coordinated agents sharing    │
│  context, escalation, memory, tools     │
└──────────────┬──────────────────────────┘
               │ "graduation"
               ▼
┌─────────────────────────────────────────┐
│        Agent Launcher (entry layer)     │
│  Single agent per role                  │
│  Uncoordinated, isolated memory         │
│  Same agent architecture as Agentic    │
│  Same tool connectors                    │
│  Signup → connect tools → deploy role  │
└─────────────────────────────────────────┘
```

**Key architectural point:** Agent Launcher agents are the same software as Agentic agents — just deployed in single-instance mode. When a customer outgrows Agent Launcher (needs 3+ coordinated agents), they upgrade to Agentic. This is intentional product ladder design.
